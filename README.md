# imgs2asciianimation
 convert jpg images into ascii animation
 
 把jpg图片转换为 ascii 字符动画

dependency / 依赖 
==========
1. 需要安装 jp2a 链接为 https://csl.name/jp2a/


执行:

    $ pip install git+https://github.com/efvhi/imgs2asciianimation.git
